# API Reference

## Endpoints

The application exposes a single server-side API endpoint via Next.js Route Handlers.

---

### `POST /api/chat`

Sends a conversation to the Gemini AI tutor and receives a streamed response.

**Location:** `app/api/chat/route.ts`

#### Request

| Field | Type | Required | Description |
|---|---|---|---|
| `messages` | `Message[]` | ✅ | Array of conversation messages (user and assistant turns) |
| `userId` | `string` | ✅ | Firebase Auth UID of the current user |

**`Message` interface:**

```typescript
interface Message {
  role: "user" | "assistant";
  content: string;
}
```

**Example request body:**

```json
{
  "messages": [
    { "role": "user", "content": "Teach me about quantum computing" },
    { "role": "assistant", "content": "Let's start with the basics..." },
    { "role": "user", "content": "What is superposition?" }
  ],
  "userId": "abc123xyz"
}
```

#### Response

**Success (200):**

Returns a streaming `text/plain` response. The response body is a `ReadableStream` that emits text chunks as they are generated by Gemini.

| Header | Value |
|---|---|
| `Content-Type` | `text/plain; charset=utf-8` |
| `Transfer-Encoding` | `chunked` |

The client should consume the stream using the Fetch API's `ReadableStream` reader:

```typescript
const response = await fetch("/api/chat", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ messages, userId }),
});

const reader = response.body?.getReader();
let fullText = "";

while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  const chunk = new TextDecoder().decode(value);
  fullText += chunk;
  // Update UI with each chunk for real-time streaming
}
```

**Validation Error (400):**

```json
{
  "error": "Validation Error",
  "details": "Missing required fields (messages or userId)"
}
```

**Server Error (500):**

```json
{
  "error": "Internal Server Error",
  "details": "Error message string"
}
```

---

#### Post-Stream Processing

After the full response has been streamed to the client, the server performs **mastery detection** on the complete response text. This is an automatic background process:

1. The full response is scanned for the regex pattern: `\[MASTERY: (.*?)\]`
2. If a match is found, the concept name is extracted
3. The concept is persisted to Firestore at: `users/{userId}/progress/mastery`
   - If the document doesn't exist, it is created with `{ mastered: [concept], lastUpdated: serverTimestamp() }`
   - If it exists, the concept is appended via `arrayUnion(concept)`

This processing happens server-side and does not affect the streamed response to the client.

## Internal Services

### `aiService.ts`

Located at `services/aiService.ts`. Contains the core AI business logic.

#### `getStratResponseStream(messages: Message[], userId: string)`

Creates a Gemini chat session and returns a streaming response iterator.

- **Parameters:**
  - `messages` — Full conversation history
  - `userId` — User identifier (reserved for future per-user model customization)
- **Returns:** `AsyncIterable` stream from Gemini's `sendMessageStream`
- **Behavior:**
  - Initializes a Gemini model with the `SYSTEM_PROMPT` from `constants/prompts.ts`
  - Converts `Message[]` to Gemini's history format (`{ role: "user" | "model", parts: [{ text }] }`)
  - Sends the latest user message and returns the streaming response

#### `handleMasteryDetection(fullResponse: string, userId: string)`

Scans a completed AI response for mastery tags and persists to Firestore.

- **Parameters:**
  - `fullResponse` — Complete text of the AI's response
  - `userId` — Firebase Auth UID
- **Returns:** `string | null` — The mastered concept name, or `null` if none detected
- **Regex Pattern:** `/\[MASTERY: (.*?)\]/`

## Firestore Data Model

### Collection: `users/{userId}/progress/mastery`

| Field | Type | Description |
|---|---|---|
| `mastered` | `string[]` | Array of concept names the user has mastered |
| `lastUpdated` | `Timestamp` | Server timestamp of the last mastery event |

**Example document:**

```json
{
  "mastered": ["Quantum Computing", "Binary Search", "Neural Networks"],
  "lastUpdated": "2026-04-25T08:30:00Z"
}
```
